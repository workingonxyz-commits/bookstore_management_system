# Bookstore Management System

A full-stack web application to manage bookstore operations including user authentication, book inventory, and analytics dashboard.

---

## Features

- User Signup and Login  
- Add, Edit, Delete Books (CRUD operations)  
- Search and Filter Books  
- Dashboard with Charts (Category-wise analysis)  
- Low Stock Alert  
- Revenue Calculation  


## Technology Stack

### Frontend
- React.js  
- Axios  
- Bootstrap  
- Chart.js  

### Backend
- Spring Boot  
- Spring Data JPA  
- Hibernate  
- REST APIs  

### Database
- MySQL  

---
## Project Structure

BOOKSTORE/  
├── demo/        (Spring Boot Backend)  
├── frontend/    (React Frontend)  

---
## How to Run

### Backend (Spring Boot)

cd demo  
mvn spring-boot:run  

Runs on:  
http://localhost:8080  



### Frontend (React)

cd frontend  
npm install  
npm start  

Runs on:  
http://localhost:3000
